# EmbOSTracker
UAV Object tracker

# Описание
EmbOSTracker (Embedding Observation-Centric Tracker) — это алгоритм для трекинга объектов, предназначенный для бесперебойного отслеживания беспилотных летательных аппаратов (БПЛА) в условиях перекрытий и окклюзий. Решение сохраняет идентификаторы объектов даже при временной потере их видимости и обеспечивает точное восстановление треков при возвращении объектов в поле зрения камеры.

# Основные возможности
Сохранение идентификаторов объектов при их взаимных перекрытиях
Восстановление трекинга после длительных окклюзий (до 2.5 секунд)
Адаптация к резким маневрам беспилотников
Работа с объектами различного масштаба в зависимости от расстояния
Поддержка нескольких одновременно движущихся БПЛА
Оптимизация для работы на embedded-устройствах

# Технологический стек
Детектор объектов: YOLOv8m
ReID модуль: EfficientNetV2-S backbone
Фреймворк: TensorFlow + KerasCV
Системные требования:
GPU: Nvidia GeForce RTX 4070 8GB (рекомендуется)
RAM: 16GB+
Операционная система: Ubuntu 20.04 LTS / Windows 10+

bash
1

Для запуска в режиме реального времени с веб-камеры:

bash
1

Доступные параметры:

123456
--video_path      Путь к видео файлу (если не указан camera_id)
--camera_id       ID камеры для режима реального времени
--output_path     Путь для сохранения результата
--conf_threshold  Порог детекции (по умолчанию 0.5)
--iou_threshold   Порог IOU для трекинга (по умолчанию 0.3)
--show            Показать результат в реальном времени (True/False)
